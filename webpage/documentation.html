The following Glyph code calculates the factorial of a number using a while loop. The code initializes a variable n to 5 and calculates the factorial of n using a while loop. The result is stored in the variable fact and returned at the end of the program. 

                
                    main = {
                        n: int32 = 5;
                        fact: int32 = 1;
                        while n > 1 {
                          fact = fact * n;
                          n = n - 1;
                        }
                        return fact
                      }

The Fibonacci sequence is a series of numbers in which each number is the sum of the two preceding ones, usually starting with 0 and 1. In the following Glyph code, the Fibonacci sequence is calculated using a recursive function. 

                
                    evoke fib(n: int32) -> int32 {
                        if n <= 1 {
                          return n
                        } else {
                          return fib(n - 1) + fib(n - 2)
                        }
                      }
                      
                      main = fib(6)

The following Glyph code describes a function that checks if a given set of numbers is a Pythagorean triple. A Pythagorean triple consists of three positive integers a, b, and c, such that a^2 + b^2 = c^2. The function is_pythag_triple takes three unsigned integers as input and returns a boolean value indicating whether the input is a Pythagorean triple. 

                
                    affix io@exscribe;
    
                    /@ numeric types can be partially genericized by dropping their size value @/
                    evoke is_pythag_triple (a, b, c: uint) -> bool 
                        = (a ** 2 + b ** 2 == c ** 2)
    
                    main = {
                        invoke exscribe is_pythag_triple (3, 4, 5);
                        invoke exscribe is_pythag_triple (3, 4, 6);
                    }
